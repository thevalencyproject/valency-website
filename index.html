The Network

Currently in development, this is a peer-to-peer, decentralised network built to be as private and efficient as possible.

The Valency Network utilises the following technologies:

  • Traceable Ring Signatures
  • Stealth Addresses
  • Quantum Proof Encryption
  • Hybrid Consensus Mechanism
  • Sharding
  • Containerisation
  • Onion Routing

The Exchange

The Valency Exchange does the following differently from other exchanges:

  • 100% Open Source and Transparent
  • Run on Smart Contracts
  • Supports FIAT currency like USD, EUR, etc.
  • Supports Decentralised Asset Trading - Contracts and NFT's
  • Can Facilitate both Peer-to-Peer and Peer-to-Exchange Transactions
